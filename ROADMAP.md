# CodeOrbit AI — Version & Capability Roadmap

This roadmap outlines the milestones and capability upgrades planned for future releases.

---

## 🚀 Version 1.0 (Release Candidate) — "Observable Foundation"
* [x] **Consensus Engine**: Multi-agent loops with Tech Lead arbitration.
* [x] **Secure Containers**: Sandboxed ast evaluations with strict RAM (512MB) limits.
* [x] **EEME Semantic Conventions**: Vector indexing for software engineering conventions.
* [x] **Developer CLI & Diagnostics**: Command line onboarding and health checks.

---

## ⚡ Version 1.1 — "Cluster Scaling & Advanced Telemetry"
* [ ] **Distributed Event Broker**: Migrating in-memory WS subscriptions to Redis or Kafka.
* [ ] **Git Tree Interactive Visualizer**: Direct file comparison diffs inside the Repository Explorer.
* [ ] **LLM Cost Budgets**: Real-time spending limits and automatic agent throttles.

---

## 🧠 Version 1.2 — "Cross-Repository Context"
* [ ] **EEME Cross-Repo Linking**: Resolving shared dependencies across microservice repositories.
* [ ] **Auto-Tuning LLM Failover**: Smart model failovers based on task difficulty.
